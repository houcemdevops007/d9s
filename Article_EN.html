d9s - Docker TUI (Terminal User Interface)

Author: KHLIFI HOUCEM / INGENIEUR DEVSECOPS && CLOUD

d9s is a powerful, lightning-fast, and intuitive Terminal User Interface (TUI) for end-to-end management of your Docker ecosystem. Designed specifically for DevOps, SecOps engineers, and Developers, d9s combines container lifecycle management, Docker Compose orchestration, and seamless integration with advanced security scanners (Trivy and Snyk) into a single keyboard-driven interface.

🚀 Key Features

1. Multi-Host Remote Management (TCP/Unix)

d9s is not limited to your local Docker daemon (Unix socket). It dynamically manages multiple standalone Docker endpoints using Contexts.

  • Dynamic Configuration: By tweaking ~/.config/d9s/config.json, you can save remote hosts (e.g., tcp://172.16.50.13:2375).
  • Instant Switching: Through the "CONTEXTS" left panel, users can actively switch from one host to another simply by pressing the Enter key, with zero downtime.

2. Centralized Docker Resource Management

Manage your entire Docker ecosystem flawlessly, without leaving the terminal:

  • Containers (key c): List containers. Shortcut bindings to Stop (x), Restart (r), or Remove (R or Del).
  • Interactive Shell (key S): Drop right into a container's command-line shell session instantaneously.
  • Images (key g or i): Explore and manage local images.
  • Volumes (key v) & Networks (key n): Comprehensive overview and operations.
  • Inspect (key i): Fetch clean JSON outputs for any selected resource.
  • Live Logs (key l): Tail streams of container logs.
  • Events & Stats: Real-time monitoring metrics including CPU/RAM.

3. Native Docker Compose (Projects) Integration

The "PROJECTS" panel bridges native container controls with Docker Compose projects:

  • Automatically detects Compose environments.
  • Full orchestration shortcuts:
    • u = Compose Up
    • d = Compose Down
    • p = Compose Pull
    • b = Compose Build

4. Built-in SecOps & Compliance (Trivy + Snyk)

The true strength of d9s lies in its native security tabs. Select an Image and:

  • Trivy Scan: Immediate open-source vulnerability reports.
  • Snyk Scan: Detailed analysis reporting using the industry standard Snyk tool.
  • Best Practices Engine: Heuristic evaluation checking for common flaws (running as root, massive sizes, non-LTS tags) using cross-correlated inspect data.

💻 Keyboard Shortcuts

The total TUI experience is optimized for keyboard power-users. No mouse is needed. Rely on Tab to pivot between focus panels and sub-tabs, Arrow Keys to traverse records, and / to kick off on-the-fly fuzzy searches.

🛠️ Multi-Platform Compilation (Linux & MacOS)

The repository provides a Makefile to abstract complex cross-compilation target architectures seamlessly. (Requirements: Go 1.22+)

# Build for your current local OS architecture
make build

# Build specifically for Linux (AMD64 & ARM64)
make build-linux

# Build specifically for macOS (AMD64 & ARM64)
make build-darwin

# Target everything (All Architectures)
make build-all

To install instantly:

sudo make install
# Moves the binary straight to /usr/local/bin/d9s
Note: For scanning tabs to operate natively, the host OS executing d9s must have trivy and snyk CLIs downloaded and installed into their respective $PATH.
